statsmodels.genmod.families.varfuncs.Binomial¶

class statsmodels.genmod.families.varfuncs.Binomial(n=1)[source]¶

Binomial variance function

Parameters:n (int, optional) – The number of trials for a binomial variable. The default is 1 for p in (0,1)
call()¶

Returns the binomial variance

Formulas()¶
--------
V(mu) = p * (1 - p) * n
where p = mu / n

Notes

Alias for Binomial: binary = Binomial()

A private method _clean trims the data by machine epsilon so that p is in (0,1)

deriv(mu) Derivative of the variance function v’(mu)
